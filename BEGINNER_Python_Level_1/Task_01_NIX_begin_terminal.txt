Написать Linux-команды для следующих вопросов:

1. Как посмотреть содержимое текстового файла?
Команды cat или less.

2. Что делает команда tail? Как посмотреть?
Выводит несколько(по умолчанию 10) последних строк из файла.
tail [options] [files]
tail -n 5 NIX_begin_terminal.txt  # Выводит последние 5 строк(n=5) файла NIX_begin_terminal.txt

3. Как посмотреть текущую рабочую директорию?
Команда pwd.

4. Как сменить рабочую директорию?
Использовать команда cd [path].

5. Как перейти в родительскую директорию?
cd ..

6. Как вернуться в домашнюю директорию?
cd ~

7. Как вывести список файлов в директории?
ls

8. Как посмотреть время последнего изменения/доступа к файлу /tmp/test.txt?
stat /tmp/test.txt (stat -c '%x / %y' /tmp/test.txt)

9. Как создать новую директорию test?
mkdir

10. Как создать пустой файл?
touch file

11. Как создать файл /tmp/2mb.txt размером 2Mb?
truncate -s 2M /tmp/test.txt

12. Как узнать тип файла?
$ file file_name

13. Как переименовать файл?
$ mv file newfile

14. Как удалить файл/директорию?
rm name_file / rmdir name_directory - удаление пустой директории
rm -r name_directory - удаление директории, содержащей подкаталоги и файлы

15. Как создать символическую/жесткую ссылку на файл/директорию?
ln файл имя_ссылки  # создаётся «жёсткая» ссылка (hard link)
Жёсткие ссылки не создаются на каталог.
ln -s файл или каталог имя_ссылки  # создаётся «символьная» ссылка (symbolic link)

16. Как посмотреть размер файла?
$ ls -lh file_name

17. Как как узнать размер директории?
du -sh file_dicertory

18. Как сравнить два текстовых файла?
$ diff file_01.txt file_02.txt

19. Как посчитать количество строк в текстовом файле?
$ wc -l file_name

20. Как вывести на экран отсортированные строки текстового файла?
$ sort test.txt
